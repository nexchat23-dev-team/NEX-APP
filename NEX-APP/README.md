# NEX-APP

This folder contains the Flutter version of NEXCHAT.

## Setup

1. Install Flutter on your machine.
2. Open this folder in a Flutter-compatible editor.
3. Run `flutter pub get`.
4. Add Firebase configuration files for Android and iOS.
5. Run `flutter run` or build an APK with `flutter build apk`.

## Project structure

- `lib/main.dart`: App entry point.
- `lib/screens/`: Screens for login, chat, profile, and splash.
- `lib/widgets/`: Shared UI widgets.
- `lib/services/`: Firebase helper services.
